This is a calculator.
It's a calculator. You put things in the text field like $$1+1$$ and it will solve it for you.
You will notice that when you try to calculate things like $$1 + \frac{1}{2}$$ it will not return 1.5 but rather $$\frac{3}{2}$$
This is intentional, and this calculator will rather use exact answers, usually fractions, than decimals like 1.5
If you want a decimal, type N; in front of your expression. Like this: $$N;1+\frac{1}{2}$$ This will return your desired answer of $$1.5$$
This calculator is a work-in-progress, and not stable.
You can try entering equations like: $$3x+2=11$$but don't expect it to work. It probably will, though.
